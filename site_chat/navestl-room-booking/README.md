# NaveSTL Room Booking

## Project Overview
NaveSTL Room Booking is a web application that allows users to book rooms for specific dates. The site features a user-friendly booking form and a gallery section with a carousel for displaying room photos, built using Bootstrap 5, HTML, CSS, and JavaScript.

## Features
- Room booking functionality with date selection and guest input.
- Responsive design using Bootstrap 5.
- Image gallery with a carousel feature to showcase room photos.

## Project Structure
```
navestl-room-booking
```

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project in your preferred code editor.
3. Open `src/index.html` in a web browser to view the application.
